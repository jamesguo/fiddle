## cache

当前模块系统中的所有模块信息

```js
seajs.cache:Object
```

## resolve

利用模块系统的内部机制对传入的字符串参数进行路径解析

```js
seajs.resolve:Function
```

## require

全局的 `require` 方法，可用来直接获取模块接口

```js
seajs.require:Function
```

## data

通过 `seajs.data`，可以查看 `seajs` 所有配置以及一些内部变量的值

```js
seajs.data:Object
```
