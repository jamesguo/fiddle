[@kind](http://usejsdoc.org/tags-kind.html) 用于描述实体类别（类、模块等）：

> - `@type` 描述的是实体的数据类型

```js
/** @kind <KINDNAME> */
```

类别名：

- class
- constant
- event
- external
- file
- function
- member
- mixin
- module
- namespace
- typedef

> - 通常类别会由其他标签推导而来，不用显式标注
> - 相关标签：[@type](http://usejsdoc.org/tags-type.html)
